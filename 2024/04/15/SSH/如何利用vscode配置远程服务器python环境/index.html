老板花巨资新买了一个256核512线程的服务器用来跑实验数据！！本文将记录如何在windows操作系统下使用vscode远程连接Linux服务器进行Python开发！！

由于Vscode的安装确实比较简单，网上的教程层出不穷，因此这里跳过Vscode的安装过程，默认本地的windows电脑下已经安装好了。

远程连接

安装 SSH

由于我们是使用 SSH 连接远程服务器，因此 Windows 需要支持 SSH，检查本地电脑是否安装了ssh？输入win+R，然后输入cmd打开控制窗口

直接输入ssh，如果出现下图所示，则证明安装好了SSH

添加插件

打开Vscode，搜索Remote Development 插件并安装

安装完插件后，在侧边栏可以看到 “远程资源管理器” 图标。

添加配置文件

目的：配置 VSCode 连接远程服务器的一些基本信息。

1、点击左侧的 “远程资源管理器” 图标，点击右上角的小齿轮（设置），选择第一个 config 文件打开

2、参考下图，填写对应信息

参考上述配置，释义如下：

  • Host ：连接的主机名称，可自定义；
  • Hostname ：远程主机的 IP 地址；
  • User ：用于登录远程主机的用户名；
  • Port ：用于登录远程主机的端口，SSH 默认为 22 ；

添加完成后ctrl+s进行保存，将会看到已保存好的SSH信息

连接测试

进行连接，并确定远程服务器是Linux，Vscode会根据服务器进行配置

如果左下角如和下图所示，则表示连接完成

由于每次连接都需要输入密码，这会导致非常的麻烦，因此可以采用公钥和私钥的方式进行免密登录。具体方式可以查看这篇文章

具体来说就是将公钥存储在远程电脑的.ssh/authorized_keys文件里，私钥放在本地电脑，并在vscode的config配置中写好私钥的具体位置

Host <远程主机名称>
HostName <远程主机IP>
User <用户名>
Port
IdentityFile <私钥本地位置>
这样子就免去了每次需要输入密码的麻烦
